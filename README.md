# StayNest Hotel Booking Backend

A backend hotel booking system inspired by Airbnb, built using Spring Boot.

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- JWT Authentication
- Stripe Payment Gateway
- Maven

## Features
- User Authentication & Authorization
- Hotel Management
- Room Booking
- Dynamic Pricing
- Payment Integration
- REST APIs
- Swagger API Documentation

## Project Structure
- Controller
- Service
- Repository
- Entity
- DTO
- Security
- Strategy Pattern

## Getting Started

1. Clone the repository
2. Configure PostgreSQL
3. Update application.properties
4. Run the project using Maven or IntelliJ
